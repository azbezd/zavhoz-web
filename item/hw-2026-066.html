Описание
Деталь относится к классу USB Type-C to Gigabit Ethernet adapter, то есть к внешним сетевым картам с интерфейсом USB-C и проводным Ethernet 10/100/1000 Мбит/с. По официальной документации TP-Link для аналогичного устройства UE300C такой адаптер работает через USB 3.0, имеет один RJ45-порт и поддерживает скорость сети до 1000 Мбит/с. В качестве канонической модели для этой позиции я выбрал типовой представитель семейства USB-C Gigabit Ethernet adapters на базе RTL8153, поскольку именно такая конфигурация прямо указана в официальных спецификациях. Устройство применяется для стабильного проводного подключения к интернету, локальной сети, стриминга, работы и передачи файлов. Для части моделей поддерживается plug-and-play в популярных ОС, а конструкция обычно компактная и рассчитана на переносное использование.
Кратко
Это USB-C сетевой адаптер/переходник для подключения устройства к проводной сети Ethernet 1 Гбит/с через порт RJ45. Обычно используется с ноутбуками, планшетами и другими устройствами без встроенного Ethernet-разъёма.
