Кратко

Это Waveshare Pan-Tilt HAT — плата расширения для Raspberry Pi, которая управляет 2-осевым поворотно-наклонным механизмом камеры (pan/tilt) по I2C. Используется для дистанционного наведения камеры и простых задач слежения/наблюдения.

Описание

Плата предназначена для управления двумя сервоприводами: по оси панорамирования и по оси наклона. На борту установлен PWM-контроллер PCA9685, а также датчик освещённости TSL25911FN/TSL2581, который помогает подстраивать работу камеры под уровень света. Подключается к стандартному 40-pin разъёму Raspberry Pi и использует I2C, поэтому требует всего две сигнальные линии для управления и измерения освещённости. Типичное применение — поворотные камеры наблюдения, робототехника и учебные проекты с Raspberry Pi. В комплектной документации Waveshare есть примеры кода для Python, WiringPi и BCM2835, а также инструкции по сборке с разными модулями камер Raspberry Pi.
