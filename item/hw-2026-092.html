Описание
Для Raspberry Pi 5 microSD-карта обычно служит основным носителем для Raspberry Pi OS или другой системы. В официальной документации Raspberry Pi указано, что для карт поддерживаются режимы DDR50 и SDR104, а для официальных карт — классы скорости C10, U3, V30 и A2. Объём 64 ГБ соответствует распространённому размеру для карт Raspberry Pi SD Cards. В контексте комплекта RPi 5 это не отдельный уникальный модуль, а стандартная карта microSD, подобранная под использование с Raspberry Pi.
Кратко
Это карта памяти microSD объёмом 64 ГБ, которая используется как загрузочный и/или системный носитель для Raspberry Pi 5 и других устройств с microSD-слотом.
