Описание
Это блок питания, предназначенный для питания Raspberry Pi 3 B+ и совместимых моделей через microUSB. Для Raspberry Pi 3 Model B+ официально рекомендуется источник питания 5 В / 2.5 А, а версия 5.1 В с запасом по току подходит для стабильной работы и компенсации потерь на кабеле. У данной позиции указан выход 5.1 В 3 А, что соответствует классу питания для Raspberry Pi 3-серии и даёт дополнительный запас по нагрузке. Такие БП обычно применяются дома, в учебных проектах и встроенных системах, где нужен компактный и надёжный источник постоянного напряжения. Для этой карточки каноническую деталь я опознал как блок питания для Raspberry Pi 3B+/3B на базе спецификации 5.1 В / 3 А; характеристики сверил по официальной документации Raspberry Pi и по техкарточке магазина с параметрами изделия.
Кратко
Сетевой блок питания Gembird NPA-DC6 для Raspberry Pi 3B/3B+ с выходом 5.1 В и током до 3 А. Используется как внешний источник питания для одноплатника через microUSB.
